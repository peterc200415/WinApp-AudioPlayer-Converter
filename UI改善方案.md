# UI 改善方案

## 當前 UI 分析

當前的 UI 使用標準的 tkinter 組件，功能完整但視覺效果較為基礎。以下是詳細的改善建議。

---

## 🎨 視覺改善方案

### 1. **整體風格提升**

#### 1.1 顏色方案
- **淺色主題**：
  - 背景：`#F5F5F5` (淺灰)
  - 主色：`#2196F3` (藍色)
  - 強調色：`#FF9800` (橙色)
  - 文字：`#212121` (深灰黑)
  
- **深色主題**：
  - 背景：`#121212` (深黑)
  - 主色：`#64B5F6` (淺藍)
  - 強調色：`#FFB74D` (淺橙)
  - 文字：`#FFFFFF` (白色)

#### 1.2 字體優化
- 標題：`Microsoft YaHei UI` 或 `Segoe UI` (12-14px, bold)
- 內文：`Microsoft YaHei UI` 或 `Segoe UI` (11-12px)
- 字幕：`Microsoft YaHei` (14-16px)

#### 1.3 間距與對齊
- 組件間距：`padx=10, pady=8`
- 內邊距：`padding=5-10`
- 統一對齊方式

---

### 2. **進度條增強** ⭐ 優先實作

#### 2.1 添加時間顯示
```
[00:00] ████████████░░░░░░░░░ [03:45 / 05:30]
```

**改善點：**
- 顯示當前播放時間
- 顯示總時長
- 格式：`MM:SS`

#### 2.2 可點擊跳轉
- 點擊進度條任意位置跳轉到對應時間
- 拖動滑鼠可預覽跳轉位置

#### 2.3 樣式美化
- 使用自定義顏色（主色）
- 添加圓角效果
- 更明顯的進度指示器

---

### 3. **播放控制按鈕美化**

#### 3.1 按鈕樣式
- 使用 `ttk.Button` 或自定義樣式
- 添加 hover 效果（顏色變化、陰影）
- 統一按鈕大小（建議 35x35 像素）
- 添加 tooltip 提示

#### 3.2 布局優化
- 按鈕分組（播放控制 / 音量 / 其他）
- 使用 Frame 分組並添加分隔線

---

### 4. **播放列表優化**

#### 4.1 信息豐富化
顯示格式：
```
1. [03:45] 歌曲名稱.m4a  ✓
2. [02:30] 另一首歌.mp3
3. [04:12] 第三首歌.wav  ✓
```

**包含：**
- 序號
- 播放時長（如果可獲取）
- 檔案名稱
- 字幕狀態圖標（✓ = 有字幕）

#### 4.2 樣式改進
- 當前播放項目：**加粗 + 主色背景 + 播放圖標 ▶**
- 有字幕項目：**粗體**
- 懸停項目：**淺色背景**
- 添加滾動條樣式

#### 4.3 排序功能
- 按名稱排序
- 按時長排序
- 按日期排序

---

### 5. **字幕顯示區域美化**

#### 5.1 樣式改進
- 居中對齊文字
- 較大字體（16-18px）
- 背景色與主題協調
- 添加淡入淡出效果（可選）

#### 5.2 雙語顯示
- 如果需要，支持同時顯示原文字幕和翻譯

---

### 6. **訊息顯示區域優化**

#### 6.1 分類顯示
- **成功訊息**：綠色
- **警告訊息**：橙色
- **錯誤訊息**：紅色
- **一般訊息**：藍色/灰色

#### 6.2 自動滾動
- 新訊息自動滾動到底部
- 可選：限制最大行數，自動清除舊訊息

---

### 7. **狀態欄 / 資訊欄**

在視窗底部添加狀態欄，顯示：
- 當前播放檔案名稱
- 播放狀態（播放中/暫停/停止）
- 總檔案數 / 當前位置
- GPU 使用狀態（如果可用）

---

## 💡 技術實現建議

### 1. 使用 ttk 主題
```python
import tkinter.ttk as ttk
style = ttk.Style()
style.theme_use('clam')  # 或其他主題
```

### 2. 自定義顏色方案
```python
COLORS = {
    'bg': '#F5F5F5',
    'fg': '#212121',
    'primary': '#2196F3',
    'accent': '#FF9800',
}
```

### 3. 進度條時間顯示
在 `ProgressBar` 組件中添加時間標籤，在 `set_value()` 時更新。

### 4. 進度條點擊跳轉
綁定 `<Button-1>` 事件，計算點擊位置對應的時間比例。
